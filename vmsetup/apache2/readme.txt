Apache location: /etc/apache2
Sites:	/etc/apache2/sites-enabled
Modules: /etc/apache2/modules-enabled

You can run a2enmod a2dismod a2ensite a2dissite to configure sites and modules.

Log files:
	/var/log/apache2/error.log
	/var/log/apache2/access.log
	
The .htaccess file is located in /var/www

Also checkout the siteconfiguration for rebeccacheri in /etc/apache2/sites-enabled
